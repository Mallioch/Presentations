MSPointer* Events

var domElement = document.getElementById('element');

domElement.addEventListener('MSPointerDown', function(evt) {
  //code goes here
});

Also MSPointerCancel, MSPointerEnter, MSPointerLeave, MSPointerMove, MSPointerOut, MSPointerOver and MSPointerUp.

This is the core event for touch programming for all Internet Explorer 10+ browsers.


if(window.PointerEvent) {
    elm.addEventListener("pointerdown", foo);
} else if (window.MSPointerEvent) {
    elm.addEventListener("MSPointerDown", foo);
} else {
    elm.addEventListener("mousedown", foo);
}
